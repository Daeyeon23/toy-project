# 증상×원인 매트릭스 — 치아바타 (ciabatta)

> **목적:** B5(프랑스·이탈리아 린도우) 배치. 식빵과 실패 양상이 가장 비슷한 린도우 계열이므로 CORE 코드 재사용을 최대화하고,
> 치아바타 고유의 실패 양상(초고수분 반죽, 성형/이동 중 기공 붕괴)만 `lean-` 신규 코드로 추가한다.

## 원인 (Causes)

| 코드 | id | name | why | action | 구분 |
|---|---|---|---|---|---|
| C1 | `yeast-dead` | 이스트 문제 | 이스트가 죽었거나 뜨거운 물에 사멸하면 발효 자체가 시작되지 않아 부풀지 않습니다. | 유통기한을 확인하고 물 온도를 40℃ 이하로 맞춘 뒤 예비 발효로 활성을 확인하세요. | CORE 재사용 |
| C2 | `underproof` | 발효 부족 | 가스가 충분히 만들어지기 전에 구우면 부피가 안 나고 속이 떡지며, 팽창 압력이 표면 약한 곳을 불규칙하게 터뜨립니다. | 1차 발효를 늘려 반죽 표면에 기포가 뚜렷이 보일 때까지 기다리세요. | CORE 재사용 |
| C3 | `overproof` | 과발효 | 발효가 과하게 진행되면 반죽이 힘을 잃어 다루는 즉시 주저앉고, 효모 대사산물로 신맛이 납니다. | 반죽 부피·기포 상태를 자주 확인하고, 과하게 늘어지면 바로 성형·굽기로 넘어가세요. | CORE 재사용 |
| C4 | `cold-environment` | 저온 환경 | 반죽·실내 온도가 낮으면 발효 속도가 크게 느려져 같은 시간에 덜 부풉니다. | 26~28℃ 환경에서 발효시켜 보세요. | CORE 재사용 |
| C5 | `weak-gluten` | 반죽 개발 부족 | 치아바타는 반죽 강도(글루텐 네트워크)만으로 80%에 달하는 수분을 붙잡아야 하는데, 개발이 부족하면 다루는 동안 구조가 바로 무너집니다.[^1] | 스트레치&폴드를 30분 간격으로 3~4회 반복해 손반죽/믹싱 없이도 글루텐을 발달시켜 보세요.[^1] | CORE 재사용 |
| C6 | `oven-too-hot` | 오븐 온도가 높음 | 겉면이 속보다 훨씬 빨리 익어 겉은 타거나 두껍고 딱딱해지는 동안, 수분이 많은 속까지 열이 전달되지 못합니다. | 오븐 온도를 레시피보다 10~15℃ 낮춰 보세요. | CORE 재사용 |
| C7 | `oven-too-cool` | 오븐 온도가 낮음 | 열이 부족하면 굽는 시간이 늘어나면서 수분이 많은 속이 설익거나 색이 약해집니다. | 오븐을 충분히 예열하고 온도계로 실제 온도를 확인하세요. | CORE 재사용 |
| C8 | `excess-hydration` | 수분 과다(의도치 않은) | 레시피보다 물이 더 많으면 반죽이 팬에 퍼지듯 흐르고, 굽는 중 가스를 가두지 못해 속이 떡지게 됩니다.[^1] | 계량을 다시 확인하고, 필요하면 물을 5% 줄이거나 폴딩 횟수를 늘려 보세요. | CORE 재사용 |
| C9 | `lean-degassing-handling` | 성형/이동 중 거친 핸들링 | 치아바타 반죽은 매우 슬랙(slack)해서, 팬으로 옮기거나 자르는 과정에서 조금만 눌러도 발효로 만든 기공이 눌려 빠져나갑니다.[^2] | 덧가루/오일을 손과 작업대에 넉넉히 묻히고, 반죽을 당기지 말고 들어 옮기듯 최소한으로 다뤄 보세요. | 신규 (`lean-` 접두, 고수분 린도우 공통 재사용 예정) |

### ✅ 조합 B: S2(주저앉음) + S8(신맛) + S10(구멍 안 생김) — PEAKED

### ⚠️ 조합 C (정직한 반례): S2(주저앉음) + S6(너무 질어 성형 안 됨) + S10(구멍 안 생김) — 근접(준-FLAT)

## 결론 (판정)

**대체로 PEAKED — 단, 치아바타 고유의 근접 쌍(개발부족 ↔ 수분과다)이 하나 확인된다.**

- ✅ 조합 A·B·D는 top-1이 2점 이상 차이로 뾰족하게 갈린다.
- ✅ `sour-smell`이 과발효 계열을 가르는 핵심 판별자로 재확인된다.
- ⚠️ 조합 C처럼 "물을 많이 넣음"과 "반죽 개발 부족"은 둘 다 초고수분 반죽에서 결과가 비슷해 근접한다 — King Arthur도 "슬랙한 반죽이 퍼지지 않으면서도 다룰 수 있는 정도를 알아가는 게 연습"이라고 서술한다.[^1]

   - `id`: `q-weakgluten-vs-hydration-ciabatta`
