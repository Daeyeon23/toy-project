# 증상×원인 매트릭스 — 포카치아 (focaccia)

> **목적:** B5(프랑스·이탈리아 린도우) 배치. 치아바타와 마찬가지로 초고수분 반죽이라 CORE 코드 재사용 비중이 크지만,
> 표면 오일(올리브 오일) 처리라는 포카치아만의 변수가 있어 관련 신규 코드를 추가한다.

| 코드 | id | name | why | action | 구분 |
|---|---|---|---|---|---|
| C1 | `yeast-dead` | 이스트 문제 | 이스트가 죽었거나 뜨거운 물에 사멸하면 발효 자체가 시작되지 않아 부풀지 않습니다. | 유통기한을 확인하고 물 온도를 40℃ 이하로 맞춘 뒤 예비 발효로 활성을 확인하세요. | CORE 재사용 |
| C2 | `underproof` | 발효 부족 | 팬 안에서 충분히 부풀지 않으면 도우가 조밀하고 딱딱해지며, 딤플(손자국)을 눌러도 다시 잘 올라오지 못합니다. | 팬 발효를 늘려 반죽이 표면 전체에 뚜렷한 기포를 보일 때까지 기다리세요.[^1] | CORE 재사용 |
| C3 | `overproof` | 과발효 | 발효가 과하게 진행되면 반죽이 힘을 잃어 오븐에 넣기 전부터 주저앉고, 신맛이 뚜렷해집니다. | 팬 발효 상태를 자주 확인하고, 표면이 과하게 늘어지면 바로 오일·토핑을 올려 굽기로 넘어가세요. | CORE 재사용 |
| C4 | `cold-environment` | 저온 환경 | 반죽·실내 온도가 낮으면 발효 속도가 크게 느려져 딤플링 후에도 잘 안 부풉니다. | 26~28℃ 환경에서 발효시켜 보세요. | CORE 재사용 |
| C5 | `weak-gluten` | 반죽 개발 부족 | 글루텐 구조가 약하면 딤플링·오일 무게를 못 버텨 부풀다 주저앉고, 성형 중 기공이 쉽게 눌려 빠집니다. | 스트레치&폴드를 반복해 글루텐을 발달시킨 뒤, 충분히 프루핑된 상태에서 딤플링하세요. | CORE 재사용 |
| C6 | `oven-too-hot` | 오븐 온도가 높음 | 겉면·표면 오일이 속보다 훨씬 빨리 익어 겉은 두껍고 딱딱해지는 동안 속까지 열이 전달되지 못합니다. | 오븐 온도를 레시피보다 10~15℃ 낮춰 보세요. | CORE 재사용 |
| C7 | `oven-too-cool` | 오븐 온도가 낮음 | 열이 부족하면 굽는 시간이 늘어나 속이 설익거나 색이 약해지고, 표면 오일이 흡수·조리되지 못한 채 고입니다. | 오븐을 충분히 예열하고 온도계로 실제 온도를 확인하세요. | CORE 재사용 |
| C8 | `excess-hydration` | 수분 과다(의도치 않은) | 레시피보다 물이 더 많으면 팬 안에서 반죽이 필요 이상으로 퍼지고, 속이 떡지게 됩니다. | 계량을 다시 확인하고, 필요하면 물을 5% 줄여 보세요. | CORE 재사용 |
| C9 | `lean-degassing-handling` | 성형/딤플링 중 거친 핸들링 | 딤플링을 너무 세게·오래 눌러 가스를 다 빼면 발효로 만든 기공이 눌려 사라지고 속이 조밀해집니다.[^2] | 딤플링은 손끝으로 가볍게, 1차 발효가 끝난 뒤 짧게 1~2회만 진행해 보세요. | 신규 (`lean-` 접두, 치아바타와 공통 재사용) |
| C10 | `lean-oil-technique` | 표면 오일 양·타이밍 문제 | 오일을 너무 많이 붓거나 굽기 직전에 부으면 딤플에 고인 오일이 다 조리되지 못해 그 부분만 눅눅·기름진 채로 남습니다.[^3] | 오일은 반죽 전체에 얇게 펴 바르고, 딤플에 소량만 고이게 조절한 뒤 충분히 예열된 오븐에서 구워 보세요. | 신규 (`lean-` 접두, 포카치아 고유) |

## 결론 (판정)

**PEAKED — 치아바타에서 재사용한 판별자(신맛)와 포카치아 고유 판별자(오일 고임+저익음 조합)가 모두 잘 작동한다.**

- ✅ 조합 A·B·C 모두 top-1이 2점 이상 차이로 뾰족하게 갈린다.
- ✅ `sour-smell`, `lean-oil-pooling`이 각각 발효 계열과 오일/오븐 계열을 가르는 판별자로 작동한다.
